25. There are 130 mango and orange trees in a farm .the number of mango trees is 20 more than the number of orange trees.how many mango trees are there?


Answer:

75

Step-by-step explanation:

Let:

x = number of mango trees

y = number of orange trees

from: there are 130 mango and orange trees

x + y = 130 (Equation 1)

from: the number of mango trees is 20 more than the number of orange trees

x = y + 20 (Equation 2)

from equation 2

x = y + 20

y = x - 20 (Equation 3)

Substitute equation 3 in equation 1

x + y = 130

x + (x - 20) = 130

x + x - 20 = 130

2x - 20 = 130

2x = 130 + 20

2x = 150

x = 150/2

x = 75
