8. The ratio of santol trees to mango trees to tamarind trees in an orchard is 5:2:8. if there a 1,245 trees in an orchard, how many mango trees are there?


Answer:

There are 166 mango trees.

Step-by-step explanation:

#1 step, add all the numbers in the ratio: 5 + 2 + 8 = 15

#2 step, Divide the total number of trees by 15, so 1,245 ÷ 15 = 83

#3 step, multiply 83 to each ratio:

83 × 5 = 415

83 × 2 = 166

83 × 8 = 664

Last step: Checking:

Santol trees + mango trees + tamarind trees = 1,245

415 + 166 + 664 = 1,245

Answer:

There are 166 mango trees.

10. There are 130 mango and orange trees in a farm .the number of mango trees is 20 more than the number of orange trees.how many mango trees are there?


Answer:

75

Step-by-step explanation:

Let:

x = number of mango trees

y = number of orange trees

from: there are 130 mango and orange trees

x + y = 130 (Equation 1)

from: the number of mango trees is 20 more than the number of orange trees

x = y + 20 (Equation 2)

from equation 2

x = y + 20

y = x - 20 (Equation 3)

Substitute equation 3 in equation 1

x + y = 130

x + (x - 20) = 130

x + x - 20 = 130

2x - 20 = 130

2x = 130 + 20

2x = 150

x = 150/2

x = 75
