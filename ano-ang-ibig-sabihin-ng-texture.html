1. 1.ano ano ang uri ng textura? 2.ano ang ibig sabihin ng monophpnic texture? 3.ano ang ibid sabihin ng hamophic texture? 4.ano ang ibig sabihin ng polyphonic texture? ​


Answer:

makinis,magaspang,madulasat bako

Answer:

1. Makinis o magaspang, madulas o mabako.

2. Ang monophonic ay ang pinakasimpleng uri ng musika na binubuo ng isang melodiya o tune at karaniwang kinakanta ng isang mang-aawit o tinutunog ng isang instrumento.

3. There is a dominant voice or a voice that act as one, and it stands out from background accompaniment.

4. Polyphony ay isang uri ng musical texture, kung saan ang isang texture ay, karaniwang nagsasalita, ang paraan na melodiko, maindayog, at maharmonya aspeto ng isang komposisyon ng musika ay pinagsama upang hugis ang pangkalahatang tunog at kalidad ng trabaho.
